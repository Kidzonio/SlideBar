# How To Create a SlideBar?

* Code Below

```lua
createSlideBar('Identificator', 
  PositionBackGroundX, 
  PositionBackGroundY, 
  BackGroundWidth, 
  BackGroundHeight, 
  PositionCircleX, 
  PositionCircleY, 
  CircleWidth, 
  CircleHeight, 
  {
  backGround = {R, G, B, A}; 
  circle = {R, G, B, A};
  })
```
