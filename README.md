# How to create a SlideBar?

```lua
createSlideBar('Identificator',
BackGroundPositionX,
BackGroundPositionY,
BackGroundWidth,
BackGroundHeight,
CircleWidth,
CircleHeight,
{
    backGround = tocolor(R, G, B, A),
    circle = tocolor(R, G, B, A)
}
)
```
